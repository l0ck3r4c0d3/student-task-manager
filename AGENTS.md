# AGENTS.md

## Project Overview

This project is a console-based Student Task Manager written in C++.

## Core Concept

The application helps students manage academic tasks, deadlines, priorities, and task progress using a simple command-line interface.

## Main Features

- Add new tasks
- View all tasks
- Delete tasks
- Mark tasks as completed
- Add deadlines
- Add priorities

## Technology Rules

- Programming language: C++
- Standard: C++17
- Interface: Console application
- Use only the C++ standard library
- No external libraries for the MVP

## Coding Style

- Keep code simple and readable
- Use meaningful variable and function names
- Write comments for important logic
- Avoid unnecessary complexity
- Prefer clear structure over clever solutions
