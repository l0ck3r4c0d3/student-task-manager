# Design System

## Framework Choice

The frontend uses HTML, CSS, and Vanilla JavaScript.

No external UI libraries are required.

## Color Palette

- Primary: #2563EB
- Secondary: #64748B
- Background: #F8FAFC
- Surface: #FFFFFF
- Text: #0F172A
- Success: #16A34A
- Error: #DC2626

## Typography

- Font family: Arial, sans-serif
- Main heading: 28px, bold
- Section heading: 20px, bold
- Body text: 16px
- Button text: 14px, bold

## Spacing

- Page padding: 24px
- Card padding: 16px
- Gap between components: 12px
- Form input padding: 10px

## Component Rules

- Buttons must have rounded corners.
- Primary buttons must use the Primary color.
- Cards must use a white background and light shadow.
- Do not place two primary buttons next to each other.
- Status `Done` must be visually shown with the Success color.
- Error messages must use the Error color.
- The UI must be simple, readable, and accessible.
