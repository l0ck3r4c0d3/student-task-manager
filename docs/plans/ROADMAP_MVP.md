# MVP Roadmap

## Project Goal

Create a console-based Student Task Manager in C++ that helps students manage academic tasks, deadlines, priorities, and progress.

## Stage 1: Project Setup

Status: Complete

- Create GitHub repository
- Add README.md
- Add AGENTS.md
- Create documentation structure
- Create initial main.cpp file

## Stage 2: Basic Task Features

Status: Complete

- Create a Task structure
- Add task title
- Store tasks in a list
- Display all tasks

## Stage 3: Task Management

Status: Complete

- Delete tasks
- Mark tasks as completed
- Show task status
- Implement Mark Task as Completed using the Command Pattern

## Stage 4: Deadlines and Priorities

Status: Planned

- Add deadline field
- Add priority field
- Display tasks with deadlines and priorities

## Stage 5: MVP Finalization

Status: Planned

- Improve menu navigation
- Add input validation
- Test all features
- Clean and refactor code

## MVP Includes

- Console menu
- Add task
- View tasks
- Delete task
- Mark task as completed
- Basic deadline and priority support

## Future Releases

- Save tasks to a file
- Load tasks from a file
- Sort tasks by deadline
- Filter tasks by status
- Improve user interface

## Scope Adjustments

The "Mark task as completed" feature was implemented as a dedicated module using the Command Pattern. This keeps task completion logic separate from the console menu and prepares the codebase for future command-based features.
