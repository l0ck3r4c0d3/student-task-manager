# Project Management Approach

This project will use a hybrid project management approach that combines Kanban and iterative development. Kanban will help track tasks visually and keep the workflow simple, while iterative development will allow the project to grow step by step from a basic MVP to a more complete application.

The human developer will be responsible for architecture decisions, code review, final testing, and approving all changes. The AI assistant will support the project by generating boilerplate code, suggesting implementation ideas, helping with debugging, and assisting with documentation. All AI-generated work must be reviewed before it is added to the final project.
