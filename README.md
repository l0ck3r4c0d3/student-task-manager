# Student Task Manager

## Product Idea

Student Task Manager is a simple application designed to help students organize their academic tasks, deadlines, and study progress in one place.

## Problem

Students often manage assignments, deadlines, exams, and personal study goals across different tools or notebooks. This can lead to missed deadlines, poor planning, and unnecessary stress.

## Target Users

The main users of this product are students who want a simple and structured way to manage their academic workload.

## Core Features

- Create, edit, and delete tasks
- Set deadlines for assignments
- Mark tasks as Todo, In Progress, or Done
- Add priority levels to tasks
- Filter tasks by status or deadline
- View upcoming deadlines

## Goal of the Project

The goal of this project is to develop a functional task management system while practicing software engineering principles such as requirements analysis, version control, documentation, and iterative development.
